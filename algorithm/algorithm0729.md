### 버블 정렬
- 인접한 두 개의 원소를 비교하며 자리를 계속 교환하는 방식
- 한 단계가 끝나면 가장 큰 원소가 마지막 자리로 정렬된다. 
기준 인덱스가 필요하다.

기준 인덱스에 범위는 0 ~ N-2(몇개씩 묶어갈지에 따라 그 수로 변경) 까지

패스가 지날때마다 기준인덱스가 줄어든다.

2개 남을때까지 진행

```python
def bubbleSort(a,N):
  for i in range(N-1,0,-1):
    for j in range(0,i):
      if a[j] > a[j+1]:
        a[j], a[j+1] =a[j+1],a[j]
```
