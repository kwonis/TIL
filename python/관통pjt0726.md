### Numpy 2차원 배열 생성
- `np.arange()` 함수로 배열 생성 후 `reshape()` 를 통해 2차원 배열로 변형
- __주의사항__: reshape 의 x, y 축 곱이 배열의 개수와 맞지 않으면 에러가 발생함

0으로만 이루어진 배열생성
 np.zeros((x축 개수, y축 개수), dtype=데이터 타입)

1로만 이루어진 배열 생성
np.ones((x축 개수, y축 개수), dtype=데이터 타입)

특정 수로 이루어진 배열 생성
np.full((x축 개수, y축 개수), 값)
